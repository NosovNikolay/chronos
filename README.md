## Setup

- download or clone the repository
  
**Server**
- cd server
- yarn
- setup .env as in emaple
- yarn prepare
- yarn migrate:dev
- yarn start

**Client**
- cd client
- yarn 
- setup .env as in emaple
- yarn start
